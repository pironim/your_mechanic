# YourMechanic

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'your_mechanic'
```

And then execute:

    $ bundle


## Usage
  to inspect solution check:
```
lib/your_mechanic.rb
```
  to inspect test cases check:
```
spec/your_mechanic_spec.rb
```

  to validate functionality run
```
  bundle exec rspec 
```
